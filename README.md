# GERENCIAR_ESTOQUE
O projeto trata-se da criação de uma empresa te Telecomunicações que prestará serviços de Telecomunicações e venda de mercadorias ligadas a sua prestação de serviço. 
Inicialmente vamos criar o cadastro de produtos de estoque de mercadorias para revenda e de materiais para suporte operacional, como por exemplo aparelhos celulares, notebooks, fones de ouvido, materiais para infraestruturea de rede, switches, cabos de fibra ótica, fios, etc.
